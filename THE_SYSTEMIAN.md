# 🌐 P-Universe — The Systemian

> *Vibe Coding • Souveraineté Numérique • Économie Personnelle*

Un écosystème CLI-first construit par un seul humain, IA-assisté, sur macOS M2.  
Pas un OS commercial. Pas un projet d'entreprise. **Un environnement de vie numérique souverain.**

---

## Philosophie

- **CLI-first** — tout se passe dans le Terminal
- **Souveraineté** — mes données restent chez moi
- **Minimalisme** — un outil, une fonction, zéro superflu
- **IA locale** — Ollama/Qwen, pas de cloud imposé
- **Open source** — si possible, toujours

---

## Outils P-Universe

### ✅ Opérationnels

| Outil | Description | Stack |
|-------|-------------|-------|
| **Pmail** | Client Gmail dans le Terminal | Python |
| **Ptime** | Tracker de temps — se déclenche à chaque création d'outil | Python |
| **PIA / Prim** | Assistant IA local avec piliers thématiques (Systemian, Tech, Finance, Santé, Art, Admin, Culture) | Ollama / Qwen |
| **Pvid** | Téléchargement de vidéos en CLI | Python / yt-dlp |

### 🔧 En développement

| Outil | Description |
|-------|-------------|
| **Psou** | Gestion budget perso — tableau Sheets-like dans le Terminal |
| **Pgit** | Versionnage automatique avec push GitHub via clé SSH |
| **Pdico** | Dictionnaire opératif personnel |
| **Pkboo** | Alertes système |
| **Pbat** | Alerte niveau batterie |
| **Psec** | Outils cybersécurité personnelle |
| **Ptree** | Organigramme coloré de P-Universe |
| **Pmasto** | Publication directe sur Mastodon depuis le Terminal |
| **Pscreen** | Workflow 100% clavier : capture écran → changement d'écran → envoi image |

## Stack technique

```
OS          macOS Tahoe — Apple M2 16GB
Shell       Fish + Micro Editor
IA locale   Ollama / Qwen / Dolphin3
Container   Docker + Ubuntu 24.04
Versioning  Git local + GitHub + Codeberg
Robotique   ROS2 + TurtleSim (en exploration)
Langages    Python 3.12 • Rust (en apprentissage)
```
